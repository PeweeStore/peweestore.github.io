Pewee is different.

No user or team management

With Pewee Store, you can share your app without a team and user management.
Every custom store purpose a distribution managed by team or by user.
Pewee follow an another philosophy.

Database? No!

Database are very great tool to store everything.
But for Pewee, we thinking a File System could be autosuffisant.
All is clear and accessible easily: FTP, SSH, Samba, etc.

Easy to update! Really.

Download zip, extract it, replace file named manifest, etc.      NO !
In Pewee, simply use Pewee update tool and all are managed automaticaly !

Upload it easily !

Pewee store purpose a simple command line tool for upload your build.
You can use it in any operating system and that very simple to use !

Give me an installer !

Lazy ? or just tired of long install process ?
For Pewee, we have choose to give you and installer app.
This app allows you to install (and update) Pewee in one-click !

In the future ?

Pewee are a simple tool at this moment. But we want something better for it's project:
IDE integrations (XCode, IntelliJ, Eclipse), Shortlink, Better management tool, etc.

Need a feature ? Think a missing point on Pewee ? Contact us !

Powered by NodeJS

We believe in NodeJS. That's a powerful tool that give to JavaScript a server side approach.
Of course Ruby, Python or another language could agree but we choose NodeJS

We Open Source

Open Source community provide a lot of awesome library for made web app.
Bootstrap, Less, Jade-Lang, ExpressJS and a lot of more !

Without forget Atom.io, a simple and pleasant code editor.
